# Acromantula 🕷️
Acromantula is a high-throughput HTTP reverse proxy written in Rust.

## Milestones
- [ ] Single backend relay (static backend config)
- [ ] Simple logging
- [ ] Response streaming
- [ ] Forward request timeout
- [ ] File-based backend config
- [ ] Multiple backend support
- [ ] Passive health markings on failure
- [ ] Rate limiting -- per IP token bucket
- [ ] Caching -- TTL support
- [ ] Metrics endpoint
  - [ ] Cache hit ratio
  - [ ] latency
- [ ] Graceful shutdown
